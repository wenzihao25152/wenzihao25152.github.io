数学

勾股定理
a² + b² = c²
在直角三角形中，两条直角边的平方和等于斜边的平方。这是几何学中最基本的定理之一，有超过400种证明方法。
几何 三角形 定理
二次方程求根公式
x = [-b ± √(b² - 4ac)] / 2a
解一般形式的一元二次方程 ax² + bx + c = 0 的公式。判别式Δ = b² - 4ac决定根的个数和性质。
代数 方程 求根

物理

牛顿第二定律
F = ma
物体的加速度与作用力成正比，与质量成反比，方向与作用力方向相同。这是经典力学的基础定律之一。
力学 牛顿 运动定律
爱因斯坦质能方程
E = mc²
能量和质量是等价的，可以相互转换。c为光速，这一公式揭示了原子能的巨大潜力，是狭义相对论的核心。
相对论 质能转换 爱因斯坦

化学

理想气体状态方程
PV = nRT
描述理想气体状态的方程，P为压强，V为体积，n为物质的量，T为温度，R为理想气体常数(8.314 J/(mol·K))。
气体 状态方程 物理化学
阿伏伽德罗定律
V₁/n₁ = V₂/n₂ (同温同压)
在相同温度和压强下，相同体积的任何气体含有相同数目的分子。1摩尔任何气体在标准状况下体积约为22.4L。
气体 摩尔 阿伏伽德罗

生物

中心法则
DNA → RNA → 蛋白质
遗传信息从DNA传递给RNA，再传递给蛋白质的规律。DNA通过转录生成RNA，RNA通过翻译生成蛋白质，这是分子生物学的核心概念。
遗传学 分子生物学 克里克
哈迪-温伯格平衡
p² + 2pq + q² = 1
在一个理想群体中，等位基因频率和基因型频率在世代传递中保持不变。p和q分别表示两个等位基因的频率，p+q=1。
群体遗传学 进化 平衡定律
米氏方程
v = (Vmax[S])/(Km + [S])
描述酶促反应速率与底物浓度关系的方程，Vmax为最大反应速率，Km为米氏常数，[S]为底物浓度。
酶动力学 生物化学 米凯利斯
光合作用总反应式
6CO2 + 6H2O → C6H12O6 + 6O2
绿色植物利用光能将二氧化碳和水转化为有机物和氧气的过程。光反应阶段将光能转化为ATP和NADPH，暗反应阶段(卡尔文循环)利用这些能量固定CO2。
植物生理 能量转换 卡尔文
